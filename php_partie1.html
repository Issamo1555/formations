1️⃣ Les Variables

Qu'est-ce qu'une variable ?
C'est une boîte qui stocke une information en mémoire. En PHP, une variable commence toujours par le symbole $.

Syntaxe de base :

<?php // Déclaration de variables avec différents types // Le symbole $ indique une variable en PHP $nom = "Dupont"; // Chaîne de caractères (string) $age = 25; // Nombre entier (int) $prix = 19.99; // Nombre décimal (float) $estActif = true; // Booléen (bool) - vrai ou faux $taille = null; // Valeur nulle // Affichage des variables echo "Bonjour " . $nom; // Le point . permet de concaténer ?>

Règles importantes :

  • Une variable commence toujours par $
  • Le nom est sensible à la casse ($Nom ≠ $nom)
  • Le nom doit commencer par une lettre ou un underscore
  • Nommez clairement vos variables pour la lisibilité
  • Pas d'espaces dans les noms de variables

2️⃣ Les Types de Données

PHP est un langage faiblement typé : le type est déterminé automatiquement selon la valeur assignée.

Tableau des types principaux :

Type Exemple Description Valeur par défaut
int $age = 30; Nombre entier 0
string $nom = "Alice"; Chaîne de caractères "" (chaîne vide)
float $prix = 9.99; Nombre décimal 0.0
bool $valid = true; Booléen (vrai/faux) false
array $notes = [15, 14]; Tableau array()
null $var = null; Aucune valeur null

Exemples de tableaux :

<?php // Tableau indexé (indices numériques) $etudiants = ["Alice", "Bob", "Charlie"]; echo $etudiants[0]; // Affiche "Alice" // Tableau associatif (clés personnalisées) $notes = [ "Alice" => 15, "Bob" => 14, "Charlie" => 16 ]; echo $notes["Alice"]; // Affiche 15 // Tableau multidimensionnel $classe = [ ["nom" => "Alice", "note" => 15], ["nom" => "Bob", "note" => 14] ]; ?>

3️⃣ Les Conditions

Les structures conditionnelles permettent d'exécuter du code différemment selon certaines conditions.

if / elseif / else :

<?php // Structure de base if/else $age = 17; if ($age < 18) { echo "Vous êtes mineur"; } elseif ($age == 18) { echo "Tout juste majeur !"; } else { echo "Vous êtes majeur"; } // Opérateurs de comparaison : // == égal à // === identique à (type et valeur) // != différent de // < inférieur à // > supérieur à // <= inférieur ou égal // >= supérieur ou égal // Opérateurs logiques : // && ET logique // || OU logique // ! NON logique $note = 15; $present = true; if ($note >= 10 && $present == true) { echo "Admis !"; } ?>

switch / case :

<?php // Alternative à if/elseif pour tester une variable // contre plusieurs valeurs possibles $note = 15; switch ($note) { case 20: echo "Parfait !"; break; // Important pour sortir du switch case 15: echo "Très bien"; break; case 10: echo "Passable"; break; default: // Cas par défaut si aucune valeur ne correspond echo "À améliorer"; } ?>

4️⃣ Les Boucles

Les boucles permettent de répéter un bloc de code plusieurs fois.

Boucle for (quand on connaît le nombre de tours) :

<?php // Structure : for (initialisation; condition; incrément) // La boucle s'exécute tant que la condition est vraie // Exemple 1 : Compteur simple for ($i = 1; $i <= 5; $i++) { echo "Tour numéro $i<br>"; } // Exemple 2 : Table de multiplication de 7 $nombre = 7; for ($i = 1; $i <= 10; $i++) { $resultat = $nombre * $i; echo "$nombre x $i = $resultat<br>"; } // Incréments/décréments : // $i++ : incrémente de 1 (post-incrément) // ++$i : incrémente de 1 (pré-incrément) // $i += 2 : incrémente de 2 // $i-- : décrémente de 1 ?>

Boucle while (tant qu'une condition est vraie) :

<?php // La condition est testée AVANT chaque tour $compteur = 0; while ($compteur < 5) { echo "Tour n°$compteur<br>"; $compteur++; // Important : modifier la variable pour éviter une boucle infinie ! } // Exemple pratique : somme des nombres de 1 à 10 $somme = 0; $i = 1; while ($i <= 10) { $somme += $i; // Équivalent à $somme = $somme + $i $i++; } echo "Somme : $somme"; // Affiche 55 ?>

Boucle do...while (exécution garantie au moins une fois) :

<?php // Le code est exécuté AVANT le test de la condition $i = 1; do { echo "Valeur : $i<br>"; $i++; } while ($i <= 5); // Même si $i = 10 au départ, le code s'exécute une fois ?>

Boucle foreach (spéciale pour les tableaux) :

<?php // Parcours simplifié des tableaux $notes = [15, 14, 16, 12]; // Version simple foreach ($notes as $note) { echo "Note : $note<br>"; } // Version avec clé/index foreach ($notes as $index => $note) { echo "Note n°$index : $note<br>"; } // Avec tableau associatif $etudiant = ["nom" => "Alice", "age" => 20]; foreach ($etudiant as $cle => $valeur) { echo "$cle : $valeur<br>"; } ?>

5️⃣ Connexion à MySQL avec PDO

PHP supporte deux extensions principales pour travailler avec des bases de données MySQL : MySQLi (MySQL Improved) et PDO (PHP Data Objects).

Devrais-je utiliser MySQLi ou PDO ?
PDO fonctionnera sur 12 systèmes de bases de données différents, tandis que MySQLi ne fonctionnera qu'avec les bases de données MySQL. Donc, si vous devez changer votre projet pour utiliser une autre base de données, PDO rend le processus facile. Les deux prennent en charge les requêtes préparées (Prepared Statements) qui sont importantes pour la sécurité des applications web.

Ouvrir une connexion à MySQL (Syntaxe PDO) :

<?php $servername = "localhost"; $username = "username"; $password = "password"; try { $conn = new PDO("mysql:host=$servername;dbname=myDB", $username, $password); // configurer le mode d'erreur PDO pour générer des exceptions $conn->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION); echo "Connected successfully"; } catch(PDOException $e) { echo "Connection failed: " . $e->getMessage(); } ?>

Astuce : Un grand avantage de PDO est qu'il possède une classe d'exception pour gérer tous les problèmes pouvant survenir dans notre base de données. Si une exception est levée à l'intérieur du bloc try{ }, le script arrête son exécution et va directement au premier bloc catch(){ }.

🛠️ TRAVAUX DIRIGÉS (TD)

✅ TD 1 : Vérifier l'âge (mineur/majeur)

Objectif : Utiliser les structures conditionnelles

Consigne : Créez un programme qui détermine si une personne est mineure ou majeure en fonction de son âge.

Solution de base :

<?php // TD 1 : Vérification de la majorité // Objectif : Utiliser if/else $age = 16; // Vous pouvez modifier cette valeur pour tester if ($age >= 18) { echo "Vous êtes majeur(e)"; } else { echo "Vous êtes mineur(e)"; } ?>

Solution avancée (avec plus de catégories) :

<?php // Version améliorée avec plusieurs catégories d'âge $age = 25; if ($age < 11) { echo "Catégorie : Enfant"; } elseif ($age < 18) { echo "Catégorie : Adolescent"; } elseif ($age < 60) { echo "Catégorie : Adulte"; } else { echo "Catégorie : Senior"; } // Vérification supplémentaire if ($age >= 18) { echo "<br>✓ Vous pouvez voter"; if ($age >= 21) { echo "<br>✓ Vous pouvez conduire sans permis probatoire"; } } ?>

Pour aller plus loin :

  • Ajouter la vérification de l'âge légal pour conduire (18 ans)
  • Ajouter la catégorie d'âge (enfant, ado, adulte, senior)
  • Vérifier si la personne peut voter (18 ans)
  • Afficher un message personnalisé selon l'âge exact

✅ TD 2 : Table de multiplication

Objectif : Maîtriser les boucles for

Consigne : Affichez la table de multiplication d'un nombre donné (de 1 à 10).

Solution de base :

<?php // TD 2 : Table de multiplication // Objectif : Utiliser une boucle for $nombre = 7; // Nombre dont on veut la table for ($i = 1; $i <= 10; $i++) { $resultat = $nombre * $i; echo "$nombre x $i = $resultat<br>"; } ?>

Solution avancée (tableau complet) :

<?php // Table de multiplication complète (1 à 10) echo "<table border='1' style='border-collapse: collapse;'>"; for ($i = 1; $i <= 10; $i++) { echo "<tr>"; for ($j = 1; $j <= 10; $j++) { $produit = $i * $j; echo "<td style='padding: 5px; text-align: center;'>$produit</td>"; } echo "</tr>"; } echo "</table>"; ?>

Challenge :

  • Créer un tableau de multiplication complet (1 à 10)
  • Afficher uniquement les tables de 1 à 5
  • Demander à l'utilisateur quel nombre il veut (avec un formulaire)
  • Afficher la table dans un tableau HTML stylisé

✅ TD 3 : Calcul de moyenne

Objectif : Manipuler les tableaux et faire des calculs

Consigne : Calculez la moyenne d'un ensemble de notes et affichez la mention correspondante.

Solution de base :

<?php // TD 3 : Calcul de moyenne // Objectif : Utiliser les tableaux et les boucles $notes = [15, 14, 16, 12, 18]; $somme = 0; // Méthode 1 : Avec une boucle foreach foreach ($notes as $note) { $somme += $note; // Additionne chaque note } // Calcul de la moyenne $nombreNotes = count($notes); // Compte le nombre de notes $moyenne = $somme / $nombreNotes; // Affichage du résultat echo "Moyenne : " . number_format($moyenne, 2) . "/20<br>"; // Attribution de la mention if ($moyenne >= 16) { $mention = "Très Bien"; } elseif ($moyenne >= 14) { $mention = "Bien"; } elseif ($moyenne >= 12) { $mention = "Assez Bien"; } elseif ($moyenne >= 10) { $mention = "Passable"; } else { $mention = "Ajourné"; } echo "Mention : " . $mention; ?>

Solution avancée (avec détails) :

<?php // Version complète avec affichage détaillé $notes = [ "Mathématiques" => 15, "PHP" => 14, "Anglais" => 12, "Base de données" => 16 ]; $somme = 0; $totalCoeff = 0; echo "<h2>Relevé de notes</h2>"; echo "<table border='1' style='border-collapse: collapse; width: 100%;'>"; echo "<tr><th>Matière</th><th>Note</th></tr>"; foreach ($notes as $matiere => $note) { $somme += $note; $totalCoeff++; echo "<tr><td>" . $matiere . "</td><td>" . $note . "/20</td></tr>"; } $moyenne = $somme / $totalCoeff; echo "<tr style='font-weight: bold; background: #f0f0f0;'>"; echo "<td>MOYENNE GÉNÉRALE</td><td>" . number_format($moyenne, 2) . "/20</td>"; echo "</tr></table>"; // Validation du semestre if ($moyenne >= 10) { echo "<p style='color: green;'>✅ Semestre VALIDÉ</p>"; } else { echo "<p style='color: red;'>❌ Semestre NON VALIDÉ</p>"; } ?>
