# Formation Certif

Projet web d'exemples PHP et HTML pour un TD de panier et d'applications.

## Contenu
- Pages HTML interactives
- Exemples PHP pour panier d'achat
- Scripts de démonstration

## Installation
1. `git clone https://github.com/Issamo1555/formations.git`
2. Ouvrir le dossier dans un serveur local
3. Lancer une page depuis le navigateur

## À propos
Ce dépôt contient des ressources de formation SmartCodai pour le développement web, incluant des projets en PHP, Python et interface utilisateur.
