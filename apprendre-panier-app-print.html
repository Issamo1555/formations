🛒 Gestion du Panier

Description: Scripts pour ajouter et gérer les articles dans le panier avec persistance via sessions.

1. ajouter_panier.php

PHP ajouter_panier.php 
<?php
/**
 * FICHIER: ajouter_panier.php
 * ROLE: Ajouter un produit au panier
 */

session_start();

// ============================================
// VÉRIFICATION DES DONNÉES
// ============================================
if (!isset($_POST['id_produit']) || !isset($_POST['quantite'])) {
    header("Location: catalogue.php");
    exit();
}

$id_produit = (int)$_POST['id_produit'];
$quantite = (int)$_POST['quantite'];

// ============================================
// INITIALISER LE PANIER SI NÉCESSAIRE
// ============================================
if (!isset($_SESSION['panier'])) {
    $_SESSION['panier'] = [];
}

// ============================================
// AJOUTER OU METTRE À JOUR LE PRODUIT
// ============================================
if (isset($_SESSION['panier'][$id_produit])) {
    // Le produit existe déjà, on ajoute la quantité
    $_SESSION['panier'][$id_produit] += $quantite;
} else {
    // Nouveau produit dans le panier
    $_SESSION['panier'][$id_produit] = $quantite;
}

// ============================================
// REDIRECTION VERS LE CATALOGUE
// ============================================
header("Location: catalogue.php?message=Produit ajouté au panier");
exit();
?>

💾 Utilisation des Sessions

Description: Les sessions permettent de conserver les données du panier entre les pages.

supprimer_panier.php

PHP supprimer_panier.php 
<?php
/**
 * FICHIER: supprimer_panier.php
 * ROLE: Supprimer un article du panier
 */

session_start();

// Vérifier qu'un ID de produit est fourni
if (!isset($_GET['id'])) {
    header("Location: panier.php");
    exit();
}

$id_supprimer = (int)$_GET['id'];

// Supprimer le produit du panier
if (isset($_SESSION['panier'][$id_supprimer])) {
    unset($_SESSION['panier'][$id_supprimer]);
}

// Redirection vers le panier
header("Location: panier.php?message=Produit supprimé");
exit();
?>

Pourquoi utiliser les sessions ?

  • Persistance: Les données survivent aux rechargements de page
  • Utilisateur unique: Chaque visiteur a son propre panier
  • Sécurité: Données stockées côté serveur
  • Expiration: Automatique après inactivité

✅ Validation de Commande

Description: Finalisation de la commande avec calcul des frais et confirmation.

valider_commande.php

PHP valider_commande.php 
<?php
/**
 * FICHIER: valider_commande.php
 * ROLE: Traiter et confirmer la commande
 */

session_start();

// Vérifier qu'il y a des articles dans le panier
if (!isset($_SESSION['panier']) || empty($_SESSION['panier'])) {
    header("Location: catalogue.php");
    exit();
}

// Catalogue des produits
$produits = [
    1 => ["nom" => "Ordinateur Portable", "prix" => 899.99],
    2 => ["nom" => "Souris Gaming", "prix" => 49.99],
    3 => ["nom" => "Clavier Mécanique", "prix" => 129.99]
];

// Calcul du total
$total = 0;
foreach ($_SESSION['panier'] as $id => $quantite) {
    $total += $produits[$id]['prix'] * $quantite;
}

// Frais de port (exemple simple)
$frais_port = ($total > 500) ? 0 : 9.99;
$total_final = $total + $frais_port;

// Simuler l'enregistrement de la commande
$numero_commande = "CMD" . date("YmdHis") . rand(100, 999);

// Vider le panier après validation
unset($_SESSION['panier']);
?>

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Commande Validée</title>
</head>
<body>
    <h1>✅ Commande Validée !</h1>

    <p><strong>Numéro de commande:</strong> <?php echo $numero_commande; ?></p>
    <p><strong>Montant total:</strong> <?php echo number_format($total_final, 2); ?> €</p>

    <h2>Détails de votre commande</h2>
    <!-- Ici on pourrait afficher le détail des articles commandés -->

    <p>Un email de confirmation vous a été envoyé.</p>

    <a href="catalogue.php">Continuer mes achats</a>

</body>
</html>

🎯 Exercices Pratiques

Exercice 1: Ajouter une réduction

Objectif: Appliquer une réduction de 10% si le total dépasse 1000€

if ($total > 1000) {
    $reduction = $total * 0.1;
    $total_final = $total - $reduction;
    echo "Réduction appliquée: " . number_format($reduction, 2) . " €";
}

Exercice 2: Vérification du stock

Objectif: Empêcher l'ajout au panier si la quantité demandée dépasse le stock disponible

Solution proposée:

// Dans ajouter_panier.php, avant d'ajouter au panier
if ($quantite > $produits[$id_produit]['stock']) {
    header("Location: catalogue.php?error=Stock insuffisant");
    exit();
}

Exercice 3: Code promo

Objectif: Ajouter un champ pour saisir un code promo (ex: "PROMO10" = -10€)

Solution proposée:

// Dans valider_commande.php
$code_promo = isset($_POST['code_promo']) ? strtoupper(trim($_POST['code_promo'])) : '';

if ($code_promo === 'PROMO10') {
    $total_final -= 10;
    $message_reduction = "Code promo PROMO10 appliqué (-10€)";
}

Exercice 4: Historique des commandes

Objectif: Sauvegarder les commandes dans un tableau en session pour les afficher dans un historique

Solution proposée:

// Dans valider_commande.php, après calcul du total
if (!isset($_SESSION['historique_commandes'])) {
    $_SESSION['historique_commandes'] = [];
}

$commande = [
    'numero' => $numero_commande,
    'date' => date('Y-m-d H:i:s'),
    'total' => $total_final,
    'articles' => $_SESSION['panier']
];

$_SESSION['historique_commandes'][] = $commande;
